- 모드: Implementation

### 1.1 계획 (Plan)
- 작업 목표: `OrganizationPage`의 단순 `Teams` 탭을 전사 단위 `Org Units` 탭으로 전환하고, 구성원들을 동적으로 할당/제거할 수 있는 관리(Manage) 모달 UI를 구축. 조직도(Org Chart)의 가시성을 높이기 위한 Scope Filter(Max Depth, Root Node) 도입.
- 완료 기준: OrgUnitGrid에서 계층별 조직 정보 표출, MemberManagementModal에서 양방향 구성원 이동 및 상태 갱신, 조직도 필터링 기능 동작, 백엔드 연동 요구사항 업데이트.

### 1.2 실행 (Act)
- 진행 현황:
  - `OrgTree.tsx`에서 edge styling을 실선으로 통일하고 Scope Filter Panel 추가 구현.
  - `TeamGrid.tsx`를 삭제하고 계층형 조직 데이터를 소화할 수 있는 `OrgUnitGrid.tsx`로 교체.
  - `MemberManagementModal.tsx` 컴포넌트 신규 생성 및 검색/이동 로직 구현.
  - `OrganizationPage.tsx`에서 위 컴포넌트들을 통합하고 volatile state(휘발성 메모리 상태) 연결.
  - `docs/backend/frontend_integration_requirements.md` 및 `ai-workflow/memory/backend_development_roadmap.md`에 백엔드 추가 설계 필요성 문서화.
