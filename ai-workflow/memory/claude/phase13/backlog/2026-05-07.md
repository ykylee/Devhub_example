### 1.1 계획 (Plan)

- 작업 목표: ADR-0001 결정 7종을 반영한 Hydra+Kratos PoC 환경을 구성하고 Next.js `/login` 에서 OIDC code flow round-trip 1회 성공.
- sub-step 분리:
  - (a) **사용자 수동, 샌드박스 외 터미널**: `go install github.com/ory/hydra/v2/cmd/hydra@vX.Y.Z`, `go install github.com/ory/kratos/cmd/kratos@vX.Y.Z`. 정확 버전은 1단계 시작 시 latest stable 결정.
  - (b) **AI 자동화**: DB schema 생성 SQL (`devhub` 안 `hydra`, `kratos`).
  - (c) **AI 자동화**: 설정 파일 작성 — `infra/idp/hydra.yaml`, `infra/idp/kratos.yaml`.
  - (d) **AI 자동화**: Kratos identity schema JSON (`traits.email`, `traits.display_name`, `metadata_public.user_id`).
  - (e) **AI 자동화**: DevHub OIDC client 등록 스크립트.
  - (f) **검증**: hydra/kratos migrate sql → hydra serve / kratos serve → Next.js `/login` round-trip 1회 성공.
- 완료 기준: (b)~(e) 산출물 작성 + (f) round-trip 1회 성공 로그.
