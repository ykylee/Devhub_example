`dev-up.{ps1,sh}` 한 명령으로 fresh DB 에서:
1. backend-core migrate (`DEVHUB_SKIP_MIGRATE=1` 우회)
2. IdP schema apply (`DEVHUB_SKIP_IDP_MIGRATE=1` 우회) — #72
3. kratos / hydra migrate sql up — #72
4. kratos serve → 4433/4434
5. hydra serve → 4444/4445
6. OIDC client register (`devhub-frontend`, `DEVHUB_SKIP_OIDC_REGISTER=1` 우회) — #73
7. backend build → `dev-bin/backend-core.exe` → 8080 — #71
8. frontend `npm.cmd run dev` → 3000

각 단계 별 skip env 로 개별 우회 가능. 외부 인스턴스가 떠 있으면 dev-up 가 자동 감지 (#69) 하고 dev-down 은 그 인스턴스를 건드리지 않음.
