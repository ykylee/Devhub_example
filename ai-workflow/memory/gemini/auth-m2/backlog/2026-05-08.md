# 작업 로그: 2026-05-08

## 오늘 수행한 작업 (마일스톤 3: 사용자 및 조직 관리 1단계)

### 1. Sign Up (셀프 가입) 기능 구현
- **백엔드**: `POST /api/v1/auth/signup` 구현. 인사 DB(`HRDBClient`)를 통한 실명/사번 검증 및 Ory Kratos Identity 생성 연동.
- **프론트엔드**: `/auth/signup` 페이지 구현. 인사 정보 기반 가입 폼 및 개발용 Mock 데이터 가이드 추가.

### 2. 조직도(Org Hierarchy) 관리 고도화
- **백엔드**: `PUT /api/v1/organization/hierarchy` 구현. 조직 구조 일괄 업데이트 및 노드 위치 저장 지원.
- **프론트엔드**: `OrgTree` 컴포넌트의 'Save' 버튼을 백엔드와 연동하여 변경 내역 영속화.

### 3. 사용자 관리(User CRUD) 유연화
- **백엔드**: `POST /api/v1/organization/users` 핸들러 고도화. `UserType` (Human/System) 도입 및 관리자 생성 시 인사 DB 검증 생략 옵션 제공.
- **백엔드**: 관리자용 인사 정보 조회 API(`GET /api/v1/hr/lookup`) 추가.
- **프론트엔드**: `UserCreationModal` 신설 및 `MemberTable` 연동. 시스템/AI 계정 생성 지원 및 시각적 구분(배지) 추가.

### 4. 개발 생산성 도구
- 가입 페이지 및 사용자 추가 모달에 Mock 인사 데이터 가이드 및 자동 완성 기능 추가.

## 기술적 결정 사항
- **Admin Bypass**: 시스템 계정(AI, 봇) 생성을 위해 관리자 경로에서는 인사 DB 검증을 필수화하지 않음.
- **Kratos Integration**: 관리자가 사용자 생성 시 비밀번호를 입력하면 Kratos Identity를 즉시 생성하여 로그인이 가능하도록 함.
