# 상세 작업 로그 (2026-05-11)

## 수행 작업
- **OIDC 연동**: `kratos_admin_client.go`를 완전히 재작성하여 `CreateIdentity` 기능을 정규화함.
- **자동 시딩 로직**: 백엔드 `main.go`에서 서버 시작 시 Kratos와 DB 스토어를 순차적으로 호출하여 `test` 관리자 계정을 생성하도록 구현.
- **인프라 스크립트**: `dev-up.sh`가 루트 실행을 지원하도록 하고, Kratos/Hydra 설정 경로 문제를 해결함.
- **환경 설정**: `DB_URL`에 로컬 macOS 계정(`yklee`) 정보를 주입하여 실제 DB 연결에 성공함.

## 발견된 이슈
- 로그인 시도 시 브라우저에서 500 에러 발생. 백엔드 로그 확인 결과 구체적인 에러 메시지 추출을 위한 추가 분석이 필요함.
- `run_service` 함수에서 리다이렉션 경로 설정에 따른 로그 파일 위치 혼선(루트 vs 하위 디렉토리) 발생 및 수정.

## 특이 사항
- `domain.User`와 `domain.AppUser` 간의 용도 차이를 확인하고, 실제 앱 사용자 모델인 `AppUser`를 사용하도록 시딩 로직 수정 완료.
