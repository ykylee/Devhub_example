# Session Handoff: UI Redesign (Variant 2)

## 세션 요약
사용자의 요청에 따라 흰색 바탕과 파란색 톤을 사용하는 새로운 UI 테마(Variant 2)를 구현하고, 기존 다크 테마(Variant 1)를 보존하였습니다. 또한 디자인 리뷰를 위한 실시간 테마 전환 토글 버튼을 추가했습니다.

## 핵심 변경 사항
- **CSS 테마 분리**: `globals.css`에서 라이트 테마를 기본(`:root`)으로 설정하고, 기존 다크 테마를 `.theme-dark` 클래스로 격리하여 보존했습니다.
- **ThemeToggle 컴포넌트**: 화면 우측 하단에 테마를 실시간으로 전환할 수 있는 디버그 버튼을 추가했습니다. (`localStorage`를 통해 상태 유지)
- **Glassmorphism 최적화**: 테마에 따라 유리 효과(Glass card)의 배경색과 그림자 강도가 자동으로 조절되도록 수정했습니다.
- **배경 그레디언트**: 라이트 모드에 최적화된 부드러운 하늘색/Indigo 그레디언트를 적용했습니다.
