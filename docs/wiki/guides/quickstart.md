# Quickstart

## DevHub를 이해하는 가장 빠른 순서

1. Home에서 제품 원칙 확인
2. 역할별 UX 칼럼 확인
3. 시스템 관리자 권한 경계 확인
4. Tech Stack으로 구현 관점 확인

## 제품 UX 핵심

- `developer` 기본 진입: 개발 대시보드
- `manager` 기본 진입: 관리 대시보드
- `system_admin` 기본 진입: 시스템 대시보드 + 시스템 설정

시스템 운영 메뉴는 `system_admin` 권한 사용자에게만 노출됩니다.

## 참고

- 이 Quickstart는 개념 온보딩용입니다.
- 내부 구현/운영 세부는 공개 위키가 아닌 내부 문서에서 관리합니다.
